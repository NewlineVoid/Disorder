About DisorderCSS

Disorder CSS is a set of CSS files that can be configured by the user in order to get the best(and most laggy) experience of Discord.
Why is it called Disorder? Because Discord users have a mental disorder, just like me :D
Anyways..
DisorderCSS provides you with two types of CSS.

  1. The main CSS, which has core variables and default themes.
  2. Module CSS, which each contain their own features.

If you can't figure it out already, The main CSS file is the most essential part of DisorderCSS.
Whereas, Modules are optional CSS files that you can import for specific features.
This page allows you to select which modules you would like to import, and generates a CSS Snippet or a JS Snippet for you.

Modules

About Modules

Modules are optional extensions that the user can import into their client in order to enhance specific parts of the UI or Theme.
You may make a pull request to request a module, or make your own module. You may DM @newline.proj on Discord for assistance, though I do plan on making a bit of documentation in the future, not that it's actually needed since it's just CSS Variables.
